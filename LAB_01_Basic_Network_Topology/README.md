# Basic LAN Switching Topology

## Overview

This project demonstrates the implementation of a basic Local Area Network (LAN) using a Cisco 2960 switch and six end devices in Cisco Packet Tracer. The topology is designed to illustrate how a Layer 2 switch enables communication between hosts within the same network.

## Objective

The objective of this lab is to:

- Build a simple LAN using a Cisco Catalyst 2960 switch.
- Connect multiple PCs to the switch using Ethernet cables.
- Configure IPv4 addresses for all hosts within the same subnet.
- Verify end-to-end connectivity using ICMP (ping).
- Understand the role of a Layer 2 switch in forwarding Ethernet frames based on MAC addresses.
- Observe how the switch learns MAC addresses dynamically by examining the MAC Address Table.

## Network Topology

The network consists of:

- 1 × Cisco Catalyst 2960 Switch
- 6 × PCs
- Copper Straight-Through cables connecting each PC to the switch
